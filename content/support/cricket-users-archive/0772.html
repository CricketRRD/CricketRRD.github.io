Kevin,

I put up a some changes I made to the Defaults file for the config-tree
and the router-interfaces Defaults file, as well as a modified version of
the listInterfaces script, called frInterfaces that changes frame-relay
interfates from "sub-interface" to "frame-interface" and adds the host
interface ifIndex, and the DLCI.

It was written for Cricket 0.69, but you shouldn't have much work to do on
it.

You can find it at http://www.sugarat.net/software/fr-config.tar.gz.

HTH,

-Tim
