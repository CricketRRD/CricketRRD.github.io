Make sure to read the CHANGES file. You'll find what you're missing in
there. Apparantly the beginner.txt is somewhat old.

In particular, you will need perl modules:
Date::Parse
DB_File (probably requires berkeley db)

And you will need to manually compile the config tree ( ~cricket/compile )

See the CHANGES file for the details.
