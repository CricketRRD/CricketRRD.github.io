Hi,

A Cisco 2924XL is not the same type of device as a Cisco 2926. Different
code base, different size, and so forth.

For a 2924XL I would suggest using the target-type of cisco-2900xl
This pulls out CPU and Memory from all the XL switches that I've tried it on.

-Jeff
