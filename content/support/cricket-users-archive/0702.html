From: "Berg, Ivan" <iberg@montana.edu>

There is no current utility to convert a rrd data file from x data sources
to x + n sources.
Looks like you probably were collecting data from 6 items, then added a
seventh item later on. Cricket cannot save data from seven sources into a
data file meant for 6 sources.

When I need to add more sources, I just delete the old rrd data file. This
is not the best solution since you lose all of your previous data. I
thought there was a process for converting the data file over but it is not
quick and painless. Maybe someone else can help you out on that one if you
need it.
