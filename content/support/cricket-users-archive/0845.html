I've created a patch for Cricket 0.70 that displays the current path within
the config tree at the top of a page allowing quick links to the tree
hierarchy. The patch is attached below. It was created using diff version
2.7 on a Linux system (diff -u format).

To apply the patch, copy the file to your cricket directory
(/usr/local/cricket-0.70 or similar) and type:
        patch -p1< cricket-0.70-current_path.patch

 <<cricket-0.70-current_path.patch>>
