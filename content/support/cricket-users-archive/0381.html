Kilian,

You will prbably be better off creating seperate interfaces file for each
router. You can put them each in their own subdirectory under
cricket-config/router-interfaces/. Then when the web site is genreated,
you will get the same format, and all the interfaces will be there.
Else, your last serial1 interface will be the only one that you will see,
as the others will get overwritten.

-Tim
