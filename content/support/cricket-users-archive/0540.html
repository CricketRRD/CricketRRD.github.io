On Tue, Aug 24, 1999 at 11:34:46AM +0100, Kevin Fenn wrote:

> From: "Kevin Fenn" <kfenn@nortelnetworks.com>
>
> Where does cricket try to resolve its host names from? I'm using
> solaris 2.6

I think that the only hostname resolution that Cricket will do is via
the SNMP_Session library, which uses the inet_aton library call. Under
Solaris, this should use whatever you specify in /etc/nsswitch.conf.
