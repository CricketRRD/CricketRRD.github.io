Daniel Spisak wrote:
> [20-Jul-1999 13:31:55 ] target millenium-gw
> auto-base = /root/cricket/cricket-config/routers/
> auto-root = /
> auto-target-path = /
> domain = mminternet.com
> inst =
> router = millenium-gw
> short-desc = Main Cisco Router for MM Internet Los Al
> snmp-host = millenium-gw.mminternet.com
> target-type = Cisco-3600-Router

There has to be a tag somewhere in there called
"rrd-datafile". Compare the sample-config root Defaults file to your
root Defaults file. Yours probably does not have a definition of
rrd-datafile. That's at least part of your problem.

You also should probably not be running Cricket as root. It's designed
to run as it's own user, or under your userid. It is not tested
running as root.
