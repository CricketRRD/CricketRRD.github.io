jmpark wrote:
> First, Sometims I would like to have report incoming traffic as
> outgoing and vice versa, when I sitting on reverse side of the link.
> How can I configure to flip the incoming and outgoing traffic rate ?

I suppose you could redefine the ifInOctets and ifOutOctets
datasources to fetch the opposite OID. If you want to do this in the
exact same directory where you are doing it forwards, then you'll need
to define a new target type called reverse-router-interface, and then
define new datasources (reverse-ifInOctets) that does the reversal.

> Second, Is it possible to diplay current maximum value of y-axis in
> graph.

No, maxs are not supported. I got a patch a while ago that started to
add it, but I wasn't able to use it, and now I don't know what I did
with it. :( If you are the person that sent me that patch, and you
have a current version, please send it in... I'm not ignoring you, I'm
just disorganized. :(

> Third, Is it possible or any option to modify hourly display
> interval? and modify string? like this 0:00, 6:00, 8:00 ---> 0 2 4 6

That would need to be added to RRD Tool. Please talk with Tobias about
it. It may in fact already be supported to some extent... I haven't
looked at RRD that closely lately.
