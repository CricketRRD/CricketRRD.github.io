Greetings all,

I have just released the alpha versions of my Cricket configs and utils for graphing information from Netsaint.

For any that are unaware, Cricket is a high performance system for monitoring
trends in dater over time and is available at
http://www.munitions.com/~jra/cricket/ and Netsaint is a network monitoring
utility available at http://www.netsaint.org

Source for my configs and utils is available at
ftp://ftp.cybertrails.com/pub/sysadmin/netsaintcricket/netsaintcricket.tar

Anyone currently using both Netsaint and Cricket, please give this a try and
lemme know if this is of any use to y'all...
