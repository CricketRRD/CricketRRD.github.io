From: Alan Nichols <Alan.Nichols@EBay.Sun.COM>

> Is it possible to run Cricket without using Apache?
>I have a friend who says it's possible to have MRTG save its
>graphs into an ftp directory, so it must be possible with Cricket

It is possible to run Cricket with a different web server - I use
SunWebServer v2.0 and v2.1, and it works just fine.

However, Cricket does not work in the same manner as MRTG. Cricket creates
graphs dynamically when requested to do so by a web server. So Cricket
will not work without a web server.
