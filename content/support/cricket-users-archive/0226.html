Hey Dionysios J. Pantazis

seems like this is a common problem, I got burned by it also.

Not sure what exact solution is but try this.

Download, run sh configure as user, not as root
edit files under cricket-configure
run compile as user, not as root
        it should come back with something like 50 targets processed, ......
make sure permissions on ALL cricket files are for user, not root

I think I got burned by this because I was switching between user and root a
lot.
But I started over fresh, following the manual(you have to run
compile-ommitted in manual), and making sure I was logged in as user the
entire time.

Try that, it might work, good luck.
