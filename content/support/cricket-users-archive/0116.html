> didn't find how I could do it (basically, I want to have a similar graph

        
        The MAX value is stored only for the 2hr and 1day intervals (see
the rra definitions in the top level Default config), but I don't believe
there is currently a simple option or config tree entry to request the MAX
be plotted instead of the AVERAGE.

        I ended up changing the grapher.cgi source to get the MAX instead
of the AVERAGE for a few specific sources. It's not a real portable code
change at this point, as it's tied to some specific target names at this
location.
