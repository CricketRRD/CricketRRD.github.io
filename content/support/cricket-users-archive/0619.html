I want to do some "mtargets", and I have the following in a config file:

target inet-total
        mtargets = "/router-interfaces/hsv_cwix;
                                /router-interfaces/ifn_atm_savvis"
        mtargets-ops = "+"
        short-desc = "Total Internet connectivity"

I have also tried "sum()" instead of "+".

The HTML page comes up okay, but the images never come up. I checked
the web log, and I get:

[Mon Sep 27 10:12:31 1999] [notice] httpd: child pid 21056 exit signal Floating point exception (8)

when running under Apache/mod_perl. If I turn off mod_perl, I get:[Mon

Sep 27 10:11:25 1999] [error] [client 206.30.33.12] Premature end of script headers: /home/sites/site1/web/cricket/mini-graph.cgi

correct variables, but that got a "Floating point exception" as well,
even under the perl debugger.

This is all on a Cobalt RaQ2 (Linux/MIPS system) with perl 5.004_04, RRD
1.07, and Cricket 0.70.

Has anyone else seen anything like this?
