Matt Duggan wrote:
> Collector, Compile, Collect-subtrees all apear to run fine
> (generates lots of e-mail! how can I stop that?) and so forth.

Collect-subtrees should only generate mail when something is going
wrong. Check the mail for errors, and fix those. Then the mail will go
away.

> example) I can see the percentage values so it appears to be working
> but it is not generating the gifs for some reason, or not displaying
> them.

It would help to see the Debug messages to see if the image is being
made right. However, Netscape servers have a bad habit of not redirecting
stderr into the server's error log file, so it might be hard to
capture it. What you want to be doing is running the grapher.cgi
script in debug mode (search for "warn" in the first hundred lines and
change it to debug) and then look at the stderr from your cgi script.
