Glenn,

Yes, it is possible. If you were graphing only 1 datasource int eh past
for a particular rrd file, and now you added one, you just confused
rrdtool. It was expecting you to continue 'get'ing 1 datasource.
You can't add new datasources, to the best of my knowledge, so you
need to delete that rrd file and then run it so the file get's built wiht
2 datasources from the start.

-Tim
