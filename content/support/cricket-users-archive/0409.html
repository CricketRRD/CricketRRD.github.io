I'm trying to get cricket up and running and have reached the point where
I'm trying to run the collector manually to test the results. The first
time I tried I got:
# ./collector /router-interface
Can't locate RRDs.pm in @INC (@INC contains: ././lib /usr/local/lib/perl5/sun4-solaris/5.00404 /usr/local/lib/perl5 /usr/local/lib/perl5/site_perl/sun4-solaris /usr/local/lib/perl5/site_perl .) at ./collector line 28.
BEGIN failed--compilation aborted at ./collector line 28.

So I found the RRDs.pm module in the rrdtool install dir (in my case that's
/usr/local/rrdtool) and copied that to /usr/local/lib/perl5/ (which is in the
@INC list). Now I'm getting the error:
# ./collector /router-interface
Can't locate loadable object for module RRDs in @INC (@INC contains: ././lib /usr/local/lib/perl5/sun4-solaris/5.00404 /usr/local/lib/perl5 /usr/local/lib/perl5/site_perl/sun4-solaris /usr/local/lib/perl5/site_perl .) at ./collector line 28
BEGIN failed--compilation aborted at ./collector line 28.

I think this means that there's a library in the rrdtools dir somewhere that I
need to move/copy someplace else. Either that, or I need to make pathing/@INC
changes so that the correct locations are included in cricket. I'm just
not sure what need to go where next.

Any assistance would be greatly appreciated.
