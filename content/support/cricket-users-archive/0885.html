Greetings,

In the top-level Defaults file there are 2 lines there you might want to edit:
width-hint = 596
height-hint = 267

In the documentation for these(if memory serves) it states that these are used
to generate the width and height hints for the html that get generated. It also
says that if these are wrong, not only will it show up in your logs, but it
will take longer to generate and display the graphs.

You can either edit these to match or just comment them out (like I do :)

Hasta,
