From: "Michael P. Burton" <mpburton@teleport.com>

Ok - so I have a switch I'm monitoring via the following entry into
switch-ports/switches:

target 131.252.45.1
        target-type= switch-port
        inst = (1..24)
        short-desc = "XSB BET"

When I run "collector" then information is collected on the port, and
all is well - it shows the device, and shows that it's gathering stats
for each interface I stipulate, making the correct .rrd files. However,
if I delete the .rrd files, and just run "collect-subtrees normal" then
no data is collected, no .rrd files are created. Any ideas?
