Hi,

With 5 targets, to add them all together, the mtargets-ops line should
look like this:

        mtargets-ops = "+,+,+,+"

So, it's comma seperated. I have done this with targets from multiple
routers just fine.

Hope this helps some. Cricket 0.63 does have all the known bugs with the
mtargets implementation fixed. Now, if someone would just fix all the
unknown bugs...

-Jeff
