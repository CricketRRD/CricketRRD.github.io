From: David Rhodes <drhodes@deakin.edu.au>

Had a similar problem. If you examine the file grapher, the -a flag refers to arguments passed into rrdtool.
Removing the @fmt to use the default format (gif) fixed the problem for me. (If you want png you can hardcode this somewhere I guess).

i.e.
# DR: modification here. Removed -a flag from args as RRDs doesn't want it!!! :)
# my(@args) = ($imageName, @fmt, @rigid, @interlaced,
        my(@args) = ($imageName, @rigid, @interlaced,

I'd be interested in comments on this as you are the first I have heard having this problem apart from myself.

Cheers

David
