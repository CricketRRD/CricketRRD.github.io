I have modified the 'test-url' program from
the cricket-0.70/util directory to be aware of
proxy servers.

Proxies can be specified by the standard LWP
environment variables, or I have a modified
sample-config/http-performance/Defaults file
to allow defining a proxy per-target in the
'urls' file.

I have multiple firewalls that are normally
load-balanced, I made my changes to test-url to
show the speed difference between my various
routes to the same outside host.

If there's interest, I'll put my changes up on
my web site and/or submit them for inclusion in
the next official release.

Kevin Kadow
MSG.Net, Inc.

(P.S. while cricket calls this 'http-performance',
test-url can be used to graph times from FTP and
gopher servers by simply stating the URL as
'ftp://host.domain/file.txt' in the 'urls' file.)
